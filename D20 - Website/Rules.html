D20 Game Rules

Initial Setup

  • Player Order: Each player rolls a d20. Highest roll goes first. Turn order then goes from highest to lowest. If players tie, tied players roll again.
  • Character Creation: Each player starts at Level 5. Players choose a class approved by the GM. Each class has a base skill from the GM. Each character starts with one class-appropriate weapon.

Turn Structure

  • Turn Composition: Each turn (scene) gives a player one movement and one action. Actions can include attacking, using a skill, interacting with the environment, or another GM-approved action.
  • Scenes: A scene is one full turn rotation. Players in the same scene may interact with each other. After all players finish their actions, turn order continues into the next scene.

Skill Checks and Combat

Skill Checks

Roll a d20 to resolve actions. To succeed, the result must be equal to or higher than the GM-set target number.

Combat

Combat order starts from the established turn order. Enemies also roll a d20 to join the order. Combat then proceeds in rounds using that order.

Health & Damage

Health

Maximum Health = Current Level x 20

Example: Level 5 = 100 max health.

Damage

Attack Damage = Current Level x d20 roll

Example: Level 5 and roll 12 = 60 damage.

Defense

Defense Value = Current Level x defense roll

On defense, roll a d20. If Defense Value is greater than or equal to Attack Value, the attack is fully blocked. Otherwise, subtract Defense Value from Attack Value to get final damage.

Final Damage = (Attacker Level x d20) - (Defender Level x d20)

Leveling Up

  • Experience and Levels: Characters can gain up to three levels in a single scene. Level gains can come from battles, events, or artifacts.

Stats Sheet

  • Skill Points: Characters gain 2 bonus stat points at each prime-number level after Level 5.
  • Point Use: Bonus points can be assigned to Might, Vigor, Agility, Mind, and Instinct. Each point adds +1 to that stat's modifier.
  • Submission: Stats must be submitted with Save and Lock. Locked points remain fixed until new prime-level points are earned.

Inventory Management

  • Items: Each character can hold up to three items. Some items may increase this limit. Starting Items must be class-appropriate and GM-approved.

Game Rules and Endgame

  • Class and Weapon Restrictions: Weapons must suit the character's class unless the GM says otherwise. Class changes are not allowed unless the GM approves them.
  • Artifact Rules: Once the GM sets an artifact's description and effects, they do not change.
  • Starting Location: All characters start in the same story location unless the GM specifies otherwise.
  • Endgame: The game ends when the GM-defined end goal is completed, such as finishing a quest, defeating a final boss, or reaching a destination.

Miscellaneous

  • Game Master's Role: The GM sets scenes, defines challenges, and arbitrates rules. The GM has final say on all game decisions, including rule interpretation and disputes.
  • Interaction and Role-Playing: Players are encouraged to role-play and interact with the world creatively. Communication and cooperation between players improve the story experience.

Reminder: If something is not explicitly covered here, the GM decides how to resolve it.
